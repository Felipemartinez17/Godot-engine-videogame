# 🚀 Space Shooter (Galactica-Style Game)

A fast-paced, arcade-inspired **space shooter** where survival is the ultimate goal. Pilot your ship through an endless storm of incoming meteors—blast them with your laser or skillfully dodge to stay alive as long as possible.

---

## 🎮 Gameplay Overview

In this Galactica-style experience, the player controls a spaceship tasked with surviving an increasingly chaotic field of falling meteors. Each meteor varies in size and speed, creating a dynamic and unpredictable challenge.

* 🔫 **Shoot meteors** to destroy them and earn points
* 🛸 **Dodge incoming obstacles** to stay alive
* ⏱️ **Survive as long as possible**—the game gets harder over time

---

## 🧠 Scoring System

* +⏳ **Score increases over time** based on survival duration
* +💥 **+5 points** for every meteor destroyed

Your final score reflects both your endurance and your skill.

---

## ⚙️ Features

* 🎯 Smooth player controls for movement and shooting
* ☄️ Randomized meteor generation (varying size & speed)
* 📈 Increasing difficulty the longer you survive
* 💡 Simple but addictive gameplay loop
* 🖥️ Clean and responsive game interface

## 🕹️ Controls

| Action | Key / Input |
| ----------- | ----------- |
| Move Left | `←` / `A` |
| Move Right | `→` / `D` |
| Move Up | `↑` / `W` |
| Move Down | `↓` / `S` |
| Shoot Laser | `Spacebar` |
